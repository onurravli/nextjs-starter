## Next.js Starter

A starter for clean Next.js projects.

### Features

- Prisma
- Zod
- NextAuth
- TailwindCSS

### Setup

1. Clone the repo

```bash
git clone https://github.com/onurravli/nextjs-starter.git
```

2. Install dependencies

```bash
npm install
```

3. Set up environment variables

```bash
cp .env.example .env
```

4. Run the development server

```bash
npm run dev
```
